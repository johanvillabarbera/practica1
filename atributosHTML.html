Cómo modificar atributos HTML desde JavaScript

En JavaScript, podemos modificar atributos HTML de elementos del DOM utilizando el método setAttribute(). Este método toma dos argumentos: el nombre del atributo y el nuevo valor del atributo.

También podemos utilizar el método setAttribute() para añadir nuevos atributos a un elemento. Para ello, simplemente pasamos el nombre y el valor del atributo como argumentos.
