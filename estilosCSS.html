Cómo modificar estilos CSS desde JavaScript

Introducción

JavaScript es un lenguaje de programación de propósito general que se utiliza ampliamente en el desarrollo web. Además de su capacidad para manipular el DOM, JavaScript también puede utilizarse para modificar los estilos CSS de los elementos HTML.

Modificando la propiedad style

La propiedad style del elemento HTML es un objeto que contiene todas las propiedades CSS aplicadas al elemento. Podemos utilizar esta propiedad para modificar el estilo del elemento en tiempo de ejecución. Para modificar la propiedad style de un elemento, primero debemos obtener una referencia al elemento. Podemos hacerlo utilizando el método querySelector() del objeto document. Una vez que tenemos una referencia al elemento, podemos acceder a su propiedad style utilizando el operador de punto.

También podemos utilizar el método setProperty() para modificar una propiedad CSS específica. El método setProperty() toma dos argumentos: el nombre de la propiedad y el valor de la propiedad.

Utilizando el método setAttribute()

El método setAttribute() del objeto Element se utiliza para establecer el valor de un atributo del elemento. Podemos utilizar este método para establecer el valor del atributo style del elemento. El valor del atributo style debe ser una cadena que contenga las propiedades CSS aplicadas al elemento.

Conclusión

Modificar estilos CSS desde JavaScript es una habilidad útil que puede utilizarse para añadir interactividad y funcionalidad a las páginas web. Con los dos métodos que hemos aprendido, podemos modificar cualquier propiedad CSS de un elemento HTML en tiempo de ejecución.
